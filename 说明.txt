问题3：log 为什么作为参数传入，而不是在文件里直接 import？
两种方式都能工作，但传参的方式更好，原因是：如果直接 import，那你在 ffmpegService.js 里创建一个 logger，它是全局的，没有 taskId 信息：
// ❌ 如果直接导入
const { appLogger } = require('./logger')
appLogger.info('提取音频完成')  
// 输出: 2025-06-25 10:00:00 info: 提取音频完成   ← 不知道是哪个任务的
