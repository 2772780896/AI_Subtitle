# Winston 日志库

> Node.js 日志库，替代 `console.log`，支持级别、格式、多输出。

---

## 核心概念

```js
const winston = require('winston')

const logger = winston.createLogger({
  level: 'info',        // ① 级别：只记录 info 及以上
  format: ...,          // ② 格式：日志长什么样
  transports: [...]     // ③ 输出：日志写到哪
})
```

---

## ① 日志级别

```
error < warn < info < http < verbose < debug < silly
```

设 `level: 'info'` → 只记录 `info`、`warn`、`error`，`debug` 及以下被忽略。

```js
logger.error('出错了')   // ✅ 记录
logger.warn('警告')      // ✅ 记录
logger.info('信息')      // ✅ 记录
logger.debug('调试')     // ❌ 被忽略
```

---

## ② 格式（format）

### 时间戳

```js
winston.format.timestamp({ format: 'YYYY-MM-DD HH:mm:ss' })
// → 每条日志自动加 timestamp 字段
```

### 自定义模板（printf）

类似 `console.log` 的模板，返回字符串：

```js
const myFormat = winston.format.printf(({ timestamp, level, message }) => {
  return `${timestamp} ${level}: ${message}`
})
// 输出：2026-06-26 19:00:00 info: 服务器已启动
```

### 组合多个格式

```js
winston.format.combine(
  winston.format.timestamp({ format: 'YYYY-MM-DD HH:mm:ss' }),
  myFormat
)
// 先加时间戳，再用 printf 拼出来
```

### 带额外字段（metadata）

```js
// 创建时设 defaultMeta，每条日志自动带上
defaultMeta: { taskId: 'abc123' }

// 或单条传入
logger.info('完成', { taskId: 'abc123' })

// printf 中可解构使用
printf(({ timestamp, level, message, taskId }) => {
  return `${timestamp} [${taskId}] ${level}: ${message}`
})
```

---

## ③ 输出（transports）

| transport | 写到哪 | 常用配置 |
|-----------|--------|----------|
| `Console` | 控制台 | `format`（可单独设格式） |
| `File` | 文件 | `filename`（文件路径） |

```js
transports: [
  new winston.transports.Console(),
  new winston.transports.File({ filename: 'app.log' })
]
```

---

## 完整示例

```js
const winston = require('winston')

// 应用级日志（不带 taskId）
const appLogger = winston.createLogger({
  level: 'info',
  format: winston.format.combine(
    winston.format.timestamp({ format: 'YYYY-MM-DD HH:mm:ss' }),
    winston.format.printf(({ timestamp, level, message }) => {
      return `${timestamp} ${level}: ${message}`
    })
  ),
  transports: [new winston.transports.Console()]
})

// 任务级日志（带 taskId，同时写控制台 + 文件）
function createTaskLogger(taskId) {
  return winston.createLogger({
    level: 'info',
    defaultMeta: { taskId },
    format: winston.format.combine(
      winston.format.timestamp({ format: 'YYYY-MM-DD HH:mm:ss' })
    ),
    transports: [
      new winston.transports.Console({
        format: winston.format.printf(({ timestamp, level, message, taskId }) => {
          return `${timestamp} ${level}: [${taskId}] ${message}`
        })
      }),
      new winston.transports.File({
        filename: `logs/${taskId}.log`,
        format: winston.format.printf(({ timestamp, level, message, taskId }) => {
          return `${timestamp} [${taskId}] ${level}: ${message}`
        })
      })
    ]
  })
}
```

---

## 对比 console.log

| | `console.log` | Winston |
|---|---|---|
| 级别 | ❌ 无 | ✅ error/warn/info/debug |
| 时间戳 | ❌ 手动加 | ✅ 自动加 |
| 写文件 | ❌ 要自己 fs | ✅ File transport |
| 多输出 | ❌ | ✅ 同时控制台+文件 |
| 格式化 | ❌ 手动拼 | ✅ printf 模板 |
