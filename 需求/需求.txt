面试作业：基于 FFmpeg 的短视频自动字幕识别与渲染

一、任务背景
在我们日常视频处理流程中，FFmpeg 是核心的音视频处理工具。本次作业考察你调用云服务进行语音识别，并熟练使用 FFmpeg 将字幕准确、美观地嵌入视频的工程化能力。
二、任务目标
请自行选取一个内容健康、有人声的短视频（30秒~3分钟，MP4格式，中文普通话或英文），完成以下工作：
1. 使用 腾讯云 ASR （官方注册有免费使用时长）录音文件识别接口，将视频中的人声转写为带精确时间戳的字幕文件（SRT 或 ASS 格式）。
2. 使用 FFmpeg 将字幕文件按照时间戳逐句渲染到视频上，输出带字幕的完整视频。
3. 自主设计字幕的字体、颜色、位置，并在说明文档中给出你的设计思路。
三、技术栈要求（重要）
- 语音识别：腾讯云 ASR，需使用录音文件识别（支持时间戳），禁止使用无时间信息的一句话识别。
- 音视频处理与字幕渲染：必须使用 FFmpeg 命令行工具或通过 Python/Shell 等语言调用 FFmpeg 实现。
不允许使用 moviepy 等封装库直接叠加字幕，推荐使用 subtitles 滤镜（SRT/ASS）或 ass 滤镜，并自定义 force_style 参数。
- 编程语言：不限，但最终的执行入口应为一个脚本或程序，可一键完成从识别到渲染的全流程。
- 环境：需在文档中注明 FFmpeg 版本及依赖的字体文件路径。
四、详细要求
1. 语音识别与字幕生成
- 从视频中提取音频，转换为 16kHz、16bit、单声道 WAV（可使用 FFmpeg 完成）。
- 调用腾讯云 ASR 录音文件识别，获取带 词级别或句子级别时间戳 的识别结果。
- 清洗结果：合并短句、去除无意义语气词（如“嗯”“啊”），保证字幕断句自然，每条字幕时长建议 1~5 秒，单条不超过两行，每行不超过20个中文字符。
- 输出标准的 SRT 或 ASS 字幕文件，要求：
  - 时间戳与音频对齐，误差 ≤ ±200ms；
  - 文件编码为 UTF-8；
  - 保留一份原始识别结果（清洗前）以备查验。
2. 使用 FFmpeg 渲染字幕
- 编写 FFmpeg 命令（或在脚本中动态生成命令），将字幕烧录到视频上。
- 必须指定以下样式参数（通过 force_style 或 ASS 样式定义）：
  - 字体名称（如 FontName=Microsoft YaHei）
  - 字体大小（FontSize=20，根据视频分辨率适配）
  - 主要颜色（PrimaryColour=&H00B4F0F0，BGR 十六进制）
  - 描边/阴影（Outline=2, Shadow=1 等）
  - 对齐方式（Alignment=2 即底部居中，或其他）
  - 字幕位置（MarginV=30 控制距底部距离等）
- 在说明文档中写清你的样式选择，并给出 设计理由（如可读性、不遮挡人物、适配移动端小屏等）。
- 最终生成的视频编码建议使用 H.264，音频 AAC，确保兼容性。
3. 工程与文档
- 代码提交至 GitHub 公共仓库，根目录包含 README.md。
- README 必须包含：
  - 运行环境（Python/Node 版本、FFmpeg 版本及安装说明）
  - 依赖安装步骤
  - 腾讯云 SecretId/SecretKey 的配置方式（通过环境变量或配置文件，严禁硬编码）
  - 一键运行命令（例如 bash run.sh input.mp4 或 python main.py input.mp4）
  - 字幕样式设计说明（200 字左右）
  - 最终视频效果截图或下载链接
- 程序需有基本错误处理（API 重试、文件格式检查）和日志输出。
- 项目目录建议结构：
/subtitle-project
├── main.py / run.sh
├── ffmpeg_cmd.py      # 存放 FFmpeg 命令构建逻辑
├── asr_client.py      # 腾讯云 ASR 交互
├── input/             # 放置原始视频
├── output/            # 生成字幕和视频
├── fonts/             # 自定义字体文件（若有）
└── README.md
五、交付物
1. GitHub 仓库链接
2. 运行日志截图（包含 ASR 调用和 FFmpeg 输出）
3. 生成的字幕文件（.srt 或 .ass）
4. 带字幕的最终视频
5. 满足上述要求的 README.md
六、加分项（不强制要求）
- 使用 ASS 格式实现更丰富的样式（如淡入淡出动画 {\fad(300,300)}）
- 自动分析视频亮度，动态调整字幕颜色以达到最佳对比度
- 支持中英文混合字幕，且排版不乱
- 使用 FFmpeg 的 drawtext 滤镜动态绘制字幕，实现特殊效果（如逐字出现）
- 提供 Docker 镜像，一键运行
- 在文档中对比不同 force_style 参数对视觉效果的影响
